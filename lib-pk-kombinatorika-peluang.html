1) Aturan Pencacahan

Jika proses memiliki beberapa tahap berurutan, jumlah cara diperoleh dengan aturan perkalian.

Jika tahap 1 ada a cara dan tahap 2 ada b cara, total = a x b

Jika pilihan bersifat alternatif (saling lepas), gunakan penjumlahan.

total = a + b + c + ...
Tip: tentukan dulu apakah situasinya "dan" (perkalian) atau "atau" (penjumlahan).

2) Faktorial, Permutasi, Kombinasi

Faktorial

n! = n x (n-1) x (n-2) x ... x 1, dan 0! = 1

Permutasi (urutan penting)

nPr = n! / (n-r)!

Kombinasi (urutan tidak penting)

nCr = n! / (r! (n-r)!)
Kesalahan umum: memakai permutasi padahal urutan tidak dipedulikan.

3) Kasus Khusus Permutasi

Permutasi dengan unsur sama

banyak susunan = n! / (a! b! c! ...)

Contoh susunan huruf pada kata "MAMA": 4!/(2!2!) = 6 susunan.

Permutasi melingkar

(n-1)!

Dipakai untuk duduk melingkar saat rotasi dianggap sama.

4) Peluang Dasar & Kejadian Majemuk

P(A) = n(A) / n(S)
P(A atau B) = P(A) + P(B) - P(A dan B)
Jika independen: P(A dan B) = P(A) x P(B)

Selalu cek dulu apakah kejadian saling bebas (independen) atau saling memengaruhi.

5) Peluang Bersyarat

P(A|B) = P(A dan B) / P(B)

Maknanya peluang A dengan syarat B sudah terjadi.

Tanpa pengembalian

Pada pengambilan tanpa pengembalian, penyebut berubah tiap tahap.

Teknik aman: tulis ruang sampel baru setelah syarat diterapkan, baru hitung peluang.

6) Distribusi Binomial Sederhana

Untuk peluang tepat r sukses dari n percobaan independen:

P(X=r) = nCr x p^r x (1-p)^(n-r)

Contoh umum: peluang muncul tepat 3 angka saat melempar koin 5 kali.

7) Strategi Cepat Mengerjakan Soal

  • Identifikasi kata kunci: "urutan", "dipilih", "tanpa pengembalian", "minimal", "tepat".
  • Sketsa kasus kecil dulu agar pola mudah terlihat.
  • Sederhanakan faktorial sebelum dikalikan penuh.
  • Jangan lupa komplemen: P(tidak A) = 1 - P(A).
  • Untuk peluang bertahap, cek apakah denominator berubah.

8) Contoh Soal & Pembahasan

Soal 1: Dari 8 siswa dipilih ketua dan wakil. Berapa banyak cara?
urutan penting: 8P2 = 8 x 7 = 56
Jawaban: 56 cara
Soal 2: Dari 10 orang dipilih 3 orang tim inti. Berapa banyak cara?
urutan tidak penting: 10C3 = (10 x 9 x 8)/(3 x 2 x 1) = 120
Jawaban: 120 cara
Soal 3: Kotak berisi 5 merah dan 3 biru. Diambil 2 tanpa pengembalian. Peluang keduanya merah?
P = (5/8) x (4/7) = 20/56 = 5/14
Jawaban: 5/14
Soal 4: Dalam 4 lemparan koin adil, peluang tepat 2 gambar?
P = 4C2 x (1/2)^2 x (1/2)^2 = 6/16 = 3/8
Jawaban: 3/8

9) Latihan Campuran

A. Banyak susunan huruf kata "STATISTIK" adalah ...
Kunci: 9!/(3!2!2!) = 15120
B. Dari 12 peserta dipilih ketua, sekretaris, bendahara. Banyak cara ...
Kunci: 12P3 = 1320
C. Peluang minimal satu angka saat melempar 2 dadu?
Kunci: 1 - (5/6 x 5/6) = 11/36
D. Dari 7 pria dan 5 wanita dipilih 4 orang, tepat 2 wanita. Banyak cara ...
Kunci: 5C2 x 7C2 = 210
E. Dalam pengambilan 3 kartu tanpa pengembalian, peluang semua kartu merah?
Kunci: (26/52) x (25/51) x (24/50) = 2/17
Target latihan: kerjakan minimal 20 soal kombinatorika-peluang campuran agar pola soal cepat terbaca.
