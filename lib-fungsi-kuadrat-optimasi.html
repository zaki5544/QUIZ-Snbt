Fungsi Kuadrat & Optimasi

Topik ini penting untuk memahami grafik parabola, titik puncak, nilai maksimum-minimum, dan soal optimasi yang sering muncul dalam penalaran matematika SNBT. Untuk latihan 2026, fokus utamanya bukan sekadar hafal rumus, tetapi mampu memodelkan masalah kontekstual ke bentuk fungsi kuadrat.

1. Bentuk umum dan bentuk puncak

Fungsi kuadrat ditulis sebagai f(x) = ax² + bx + c dengan a ≠ 0. Grafiknya berbentuk parabola.

  • Jika a > 0, parabola terbuka ke atas.
  • Jika a < 0, parabola terbuka ke bawah.
  • Bentuk puncak sering ditulis sebagai f(x)=a(x-h)²+k, dengan titik puncak (h,k).
Contoh: f(x)=x²-4x+3 adalah parabola terbuka ke atas karena a = 1. Jika dilengkapkan kuadrat, diperoleh (x-2)²-1, sehingga titik puncaknya adalah (2, -1).

2. Titik puncak, sumbu simetri, dan akar

Titik puncak adalah bagian paling tinggi atau paling rendah dari parabola. Nilai absis titik puncak dapat dicari dengan rumus:

x = -b / 2a

Setelah memperoleh nilai x, substitusikan ke fungsi untuk mendapatkan nilai y. Selain itu, akar-akar persamaan kuadrat penting untuk mengetahui titik potong dengan sumbu-x.

D = b² - 4ac
  • D > 0: dua akar real berbeda.
  • D = 0: satu akar kembar.
  • D < 0: tidak memotong sumbu-x.
Shortcut: kalau yang ditanya hanya sumbu simetri, cukup pakai x = -b / 2a. Kalau ditanya bentuk grafik, cek juga tanda a dan diskriminannya.

3. Nilai maksimum dan minimum

Nilai optimum fungsi kuadrat berada di titik puncak.

  • Minimum terjadi jika parabola terbuka ke atas.
  • Maksimum terjadi jika parabola terbuka ke bawah.
Latihan singkat: tentukan nilai minimum dari f(x)=x²-6x+13.
Jawab: x = -(-6)/(2×1)=3, maka f(3)=4. Jadi nilai minimumnya 4.

Kenapa ini penting?

Pada soal SNBT 2026, maksimum-minimum sering muncul dalam konteks luas terbesar, keuntungan terbesar, biaya paling kecil, atau jarak paling dekat. Jadi kamu harus bisa menghubungkan nilai puncak dengan situasi cerita.

4. Optimasi kontekstual langkah demi langkah

Soal optimasi biasanya muncul dalam bentuk cerita: luas maksimum, biaya minimum, keuntungan maksimum, dan sebagainya.

  1. Tentukan variabel utama, misalnya panjang = x.
  2. Nyatakan besaran lain dalam variabel yang sama.
  3. Bentuk fungsi kuadrat dari situasi tersebut.
  4. Cari titik puncak untuk memperoleh nilai optimum.
  5. Tafsirkan kembali hasilnya ke konteks soal.
Contoh konteks: Sebuah taman persegi panjang memiliki keliling 20 m. Jika panjangnya x, maka lebarnya 10 - x. Luas taman:
L(x)=x(10-x)=-x²+10x.
Titik puncak ada di x=5, sehingga luas maksimum adalah 25 m².

5. Jebakan umum model SNBT 2026

  • Berhenti di nilai x titik puncak, padahal yang ditanya adalah luas, biaya, atau keuntungan.
  • Lupa membedakan kapan diminta nilai optimum dan kapan diminta nilai x saat optimum terjadi.
  • Salah mengubah cerita menjadi model aljabar.
  • Terlalu fokus pada hitungan, padahal soal penalaran matematika juga menilai interpretasi grafik.
Catatan relevansi: tidak ada materi yang “ketinggalan zaman” di sini untuk 2026; yang berubah biasanya hanya konteks soal. Jadi kuasai konsep, bukan hafalan template semata.

6. Latihan bertahap

Soal 1: Tentukan titik puncak dari f(x)=2x²-8x+5.

Pembahasan: x = -(-8)/(2·2)=2. Lalu f(2)=2(4)-16+5=-3. Titik puncak = (2, -3).

Soal 2: Sebuah usaha kecil memiliki keuntungan K(x)=-x²+12x-20. Berapa keuntungan maksimum?

Pembahasan: karena a < 0, fungsi memiliki nilai maksimum. x = -12/(2·-1)=6, maka K(6)=16. Jadi keuntungan maksimumnya 16.
