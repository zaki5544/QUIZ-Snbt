Persentase, Diskon & Bunga

Topik ini sangat sering muncul dalam bentuk soal konteks belanja, usaha, tabungan, kredit, dan investasi. Kuncinya adalah mengenali nilai acuan (basis persen), lalu memilih rumus yang sesuai.

1) Konsep Dasar Persentase

Persentase menyatakan bagian dari 100. Tiga bentuk yang selalu dipakai:

persentase = (bagian / total) x 100%
bagian = persentase x total
total = bagian / persentase

Kenaikan dan penurunan

  • Naik p%: nilai baru = nilai awal x (1 + p/100)
  • Turun p%: nilai baru = nilai awal x (1 - p/100)
  • Naik p% lalu turun p% tidak kembali ke nilai awal.
Kesalahan umum: siswa sering memakai persen dari nilai akhir, padahal soal meminta persen dari nilai awal.

2) Diskon, Pajak, dan Harga Bersih

Diskon tunggal

harga setelah diskon p% = harga awal x (1 - p/100)

Diskon bertingkat

Jika diskon 20% lalu 10%, maka:

harga akhir = harga awal x 0.8 x 0.9 = harga awal x 0.72

Artinya diskon total efektif 28%.

Pajak setelah diskon

Urutan operasi penting. Terapkan diskon dulu, baru pajak (jika soal demikian).

Tip SNBT: ubah semua persentase jadi faktor pengali agar perhitungan lebih cepat.

3) Untung-Rugi dan Markup

untung = harga jual - harga beli
rugi = harga beli - harga jual
% untung/rugi = (untung atau rugi / harga beli) x 100%

Markup (kenaikan dari modal)

harga jual = harga beli x (1 + markup)

Jika markup 25% dan ada diskon 10%, maka tetap hitung berurutan berdasarkan harga saat itu.

4) Bruto, Netto, Tara

Konsep ini sering muncul pada soal paket barang dan kemasan.

bruto = netto + tara
netto = bruto - tara
tara(%) = (tara / bruto) x 100%

Perhatikan basis persen untuk tara biasanya terhadap bruto, kecuali soal memberi ketentuan khusus.

5) Bunga Tunggal dan Bunga Majemuk

Bunga tunggal

A = P (1 + r x t)

Bunga dihitung dari pokok awal saja.

Bunga majemuk

A = P (1 + r)^t

Bunga periode berikutnya ikut menghitung bunga sebelumnya.

Jika periode tidak tahunan

Pastikan r dan t konsisten, misalnya r per bulan dengan t bulan.

Kesalahan umum: memakai rumus tunggal untuk soal yang jelas majemuk.

6) Strategi Hitung Cepat SNBT

  • Ubah persen ke faktor: 15% = 0.15, 125% = 1.25.
  • Gunakan bentuk pecahan yang mudah: 25% = 1/4, 12.5% = 1/8.
  • Jangan buru-buru: tentukan nilai acuan dulu.
  • Jika berantai, simpan dalam bentuk faktor berurutan lalu kalikan sekali.
  • Periksa hasil akhir masuk akal terhadap konteks (harga setelah diskon tidak mungkin naik jika tidak ada pajak/biaya tambahan).

7) Contoh Soal dan Pembahasan

Soal 1: Harga jaket Rp500.000 didiskon 20% lalu dikenai pajak 11%. Berapa harga akhir?
setelah diskon = 500000 x 0.8 = 400000 setelah pajak = 400000 x 1.11 = 444000
Jawaban: Rp444.000
Soal 2: Sebuah barang dibeli Rp240.000 dan dijual untung 15%. Berapa harga jual?
harga jual = 240000 x 1.15 = 276000
Jawaban: Rp276.000
Soal 3: Tabungan Rp3.000.000 berbunga tunggal 8% per tahun selama 2 tahun. Nilai akhir?
A = 3000000 (1 + 0.08 x 2) = 3000000 x 1.16 = 3480000
Jawaban: Rp3.480.000
Soal 4: Modal naik dari Rp1.600.000 menjadi Rp2.000.000. Berapa persen kenaikan?
kenaikan = 400000 % kenaikan = 400000 / 1600000 x 100% = 25%
Jawaban: 25%

8) Latihan Campuran

A. Harga laptop Rp8.000.000 didiskon 15% lalu 5%. Berapa harga akhir?
Kunci: Rp6.460.000
B. Barang dijual Rp345.000 dengan rugi 8%. Berapa harga belinya?
Kunci: Rp375.000
C. Uang Rp2.500.000 dibungakan majemuk 10% per tahun selama 2 tahun. Nilai akhir?
Kunci: Rp3.025.000
D. Bruto 50 kg dan tara 4%. Berapa netto?
Kunci: 48 kg
E. Sebuah nilai naik 30% lalu turun 20%. Jika awal 500, nilai akhir berapa?
Kunci: 520

9) Ringkasan dan Checklist

  • Saya bisa menentukan nilai acuan persen dengan benar.
  • Saya paham perbedaan diskon tunggal dan diskon bertingkat.
  • Saya bisa hitung untung-rugi dan markup tanpa tertukar basis.
  • Saya bisa bedakan kapan memakai bunga tunggal dan majemuk.
  • Saya bisa mengecek kewajaran jawaban akhir.
Target latihan: kerjakan minimal 15 soal campuran topik ini sebelum simulasi tryout berikutnya.
